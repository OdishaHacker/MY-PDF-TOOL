# MY PDF-TOOL

All-in-One PDF Tools — Free, Private, No Upload Required.

## Tools

- **Organize**: Merge, Split, Organize, Rotate PDF
- **Optimize**: Compress, Repair PDF
- **Convert to PDF**: JPG, Word, Excel, HTML, Text → PDF
- **Convert from PDF**: PDF → JPG, Word, Excel, PowerPoint, Text
- **Edit**: Edit, Watermark, Page Numbers, Crop, Redact
- **Security**: Protect, Unlock, Sign PDF

## Tech Stack

Next.js 16 · TypeScript · Tailwind CSS · pdf-lib · pdfjs-dist · jspdf · docx · xlsx · pptxgenjs

## Run Locally

```bash
npm install
npm run dev
```

## Deploy with Docker (Coolify)

```bash
docker-compose -f docker-compose.coolify.yml up -d
```
